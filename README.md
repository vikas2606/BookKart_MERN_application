
# BookKart

An online book store build using MERN stack


## Tech Stack

 **Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB


## Installation


Clone the project

```bash
  git clone https://github.com/vikas2606/BookKart_MERN_application.git
```

Go to the project directory

```bash
  cd BookKart
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```


Start the Client

```bash
  cd frontend
  npm run start
```
Start the server

In the root folder
```bash
  npm run start
```
